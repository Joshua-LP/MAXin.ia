## 💻 Desarrollo Local

Solo necesitas un servidor HTTP sencillo.

```bash
# Instala http-server si no lo tienes
npm install -g http-server

# Corre el servidor
http-server .
```

O si usas Python:
```bash
python -m http.server 8000
```
Abre en tu navegador `http://localhost:8000`
